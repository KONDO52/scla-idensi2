# ずぼらでダイエット デモ機

運動せずに食事管理だけで痩せたい「ずぼら層」向けのダイエットアプリのプロトタイプ。

## 🌐 デモページ

https://kondo52.github.io/scla-idensi2/

スマートフォンで開くか、PCブラウザでレスポンシブ表示してください。

チーム限定公開のため、サイト閲覧にはパスワードが必要です:


## 📝 概要

- HTMLのみで構成されたクライアントサイドデモ
- S-001〜S-080 までの全画面が含まれる（約130画面）
- 食事記録・わがまま・停滞期検知・塩分コンシェルジュ等を体験可能
- 実データはありません（全てダミー）
